# CLI

Install the package in editable mode to get the `okx-platform` command.

## Commands

- `okx-platform status`
- `okx-platform deploy BOT_NAME --profile demo`
- `okx-platform enable BOT_NAME`
- `okx-platform disable BOT_NAME`
- `okx-platform order --profile demo --inst-id BTC-USDT-SWAP --instrument-kind swap --side buy --size 1 --td-mode isolated`
- `okx-platform cancel --profile demo --inst-id BTC-USDT-SWAP --order-id ...`
- `okx-platform stop-all --reason "manual intervention"`
- `okx-platform services`
- `okx-platform profiles`
- `okx-platform instruments`
