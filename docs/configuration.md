# Configuration

## Core environment variables

- `CONTROL_API_URL`: control plane base URL for CLI and service-to-service calls
- `CONTROL_API_DATABASE_URL`: SQLAlchemy database URL
- `CONTROL_API_KEY`: API key for protected control plane routes
- `TRADING_PROFILE`: `demo` or `live`

## OKX credentials

- `OKX_DEMO_API_KEY`
- `OKX_DEMO_SECRET_KEY`
- `OKX_DEMO_PASSPHRASE`
- `OKX_LIVE_API_KEY`
- `OKX_LIVE_SECRET_KEY`
- `OKX_LIVE_PASSPHRASE`

## Reference strategy

- `REFERENCE_BOT_NAME`
- `REFERENCE_INST_ID`
- `REFERENCE_INSTRUMENT_KIND`
- `REFERENCE_TRIGGER_SPREAD`
- `REFERENCE_ORDER_SIZE`

## Optional notifications

- `SLACK_WEBHOOK_URL`
